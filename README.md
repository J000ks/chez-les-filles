# Chez les filles — Site vitrine

Site vitrine artisanal pour la vente de crêpes professionnelles en Finistère.

## Structure

```
chez-les-filles/
├── index.html              ← Page principale (toutes les sections)
├── README.md
└── assets/
    ├── images/
    │   ├── douzaine-home.jpg      ← Carte produit page d'accueil
    │   ├── caramel-home.png       ← Carte roulés page d'accueil
    │   ├── douzaine.jpg           ← Photo page Produits (crêpes classiques)
    │   ├── caramel.png            ← Photo page Produits (roulés sucrés)
    │   └── bleunoir-roule.png     ← Photo page Produits (roulés salés)
    └── videos/
        ├── hero.mp4               ← Vidéo fond page d'accueil (MP4)
        └── hero.webm              ← Vidéo fond page d'accueil (WebM, Firefox)
```

## Déploiement sur GitHub Pages

1. Créer un dépôt GitHub (ex: `chez-les-filles`)
2. Uploader **tous les fichiers** en respectant la structure ci-dessus
3. Dans le dépôt → **Settings → Pages → Branch: main → / (root)**
4. Le site est en ligne à : `https://[votre-username].github.io/chez-les-filles/`
